# Contributing to Crypto Trading Platform UI

Thank you for considering contributing!

## How to Contribute
1. Fork the repository  
2. Create a new branch  
3. Make changes  
4. Commit and push  
5. Open a Pull Request  

## Coding Rules
- Use TypeScript  
- Follow component structure  
- Format your code (`npm run lint --fix`)  

## Commit Messages
Use conventional commits:
- feat: new feature  
- fix: bug fix  
- docs: documentation  
- refactor: code cleanup
